# 🧩 Sprite Studio · Material You

A sleek, client-side sprite sheet generator built with vanilla JavaScript and Google's Material Design 3 (Material You) aesthetics. Pack individual animation frames into a single optimized PNG atlas with zero dependencies.

## ✨ Features

- **🎨 Material You Design** — Clean dark theme with dynamic elevation and smooth animations.
- **📦 Batch Processing** — Upload multiple PNG files at once.
- **⚙️ Customizable Layout** — Adjust padding and column count in real-time.
- **📝 Custom Filename** — Set the output filename directly in the UI.
- **🖨️ JSON Metadata** — Frame coordinates (`x, y, width, height`) are logged to the console for easy engine integration.
- **🚫 Zero Dependencies** — Pure HTML/CSS/JS. Works out of the box.

## 🚀 How to Use

1. Clone the repo or download `index.html`.
2. Open the file in any modern browser.
3. Upload your PNG frames.
4. Set padding, columns, and the desired filename.
5. Click **Generate** and download your sprite sheet.

## 🛠️ Tech Stack

- HTML5 Canvas API
- CSS3 (Flexbox/Grid, Material Design 3 Tokens)
- Vanilla JavaScript (ES6+)
